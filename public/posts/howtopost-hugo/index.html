Obisdianで編集した.mdファイルをGitでリモートリポジトリにpushしてHugoで変換してNetlifyでpostする流れ(自分用メモ)

Jul 6, 2021
1 min read

手順

  1. pathを指定してディレクトリを変更
  2. Hugoで新しいMarkdownファイルを作成
  3. Obsidianでファイルを編集
  4. Hugoでサイトを構築
  5. Gitでリモートリポジトリにpush

commands

cd hugo/example/
hugo new posts/example.md

hugo -D
git add .
git commit -m "commit changes"
git push -u origin main
